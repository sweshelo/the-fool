# AGENTS.md

## リポジトリ概要

このリポジトリは CODE OF JOKER のシミュレータをWebSocketサーバの形で提供するリポジトリです。
コアシステムを含む `src/package` ディレクトリと、カードの効果やそれを実現するためのヘルパー類が `src/database` に格納されます。

## コーディング規約

このリポジトリでは TypeScript を利用します。ランタイムとしては `Bun` を利用し、デプロイサービス上では `Bun` をベースとしたDockerコンテナが動きます。
ローカル環境では、現在のところ直接 Bun を利用しています。
ソースコードに対しては `oxlint` と `oxfmt` によるコードのスタイリングが実施されます。これらはGitコミット時に `lefthook` によって自動でチェックされます。

## セキュリティ

現在のところ、ユーザ固有の情報を保存・利用することはないため、一般的なWeb開発基準に則ります。また、開発中は HTTP の接続を許容する場合があります。

## ビルド・テスト

現在のところ、テストコードはありません。

ビルドに特殊な設定は不要です。

## 知識

カードの効果を実装する場合、[\_guide.md](src/database/effects/cards/_guide.md) の確認が必要です。
